## Come Creare Articoli

Creare un nuovo articolo è semplice! Basta aggiungere un nuovo file `.mdx` nella cartella `content/blog/` con i metadati corretti nel frontmatter.

## Formattazione

Puoi usare tutta la formattazione Markdown standard:

- **Grassetto** per enfasi
- *Corsivo* per citazioni
- `codice` per termini tecnici

> Le citazioni sono perfette per evidenziare frasi importanti.

## Conclusione

Il sistema MDX ti permette di avere il controllo completo sui tuoi contenuti senza bisogno di un database o un CMS esterno.
